# Мониторинг рынка машиностроения

Учебный проект по практике: сервис для сбора и анализа данных о ценах и ассортименте российских машиностроительных компаний.

Фокус категорий:
- подшипники
- насосы
- электродвигатели
- фильтры (сетчатые)

Цель проекта — собирать данные с сайтов компаний и:
- получить единую таблицу с названиями компаний, товаров и цен,
- найти **ёмкость предложения** (сколько позиций у кого),
- вычислить **уровень цен** по категориям и компаниям,
- отметить особенности и разброс цен в разных сегментах.

---

## 1. Архитектура проекта

1. **Исследование рынка**
   - выбор подкатегорий в машиностроении;
   - подбор компаний и проверка их сайтов (наличие каталога, цен, robots.txt).

2. **Сбор данных (парсер)**
   - модульный парсер под конкретные сайты;
   - конфиги с селекторами для карточек товаров;
   - выгрузка сырых данных в CSV.

3. **Обработка данных**
   - очистка текстов, удаление дублей и мусора;
   - приведение цен к числовому формату в RUB;
   - стандартизация категорий и названий товаров;
   - формирование финальной таблицы.

4. **Аналитика и визуализация**
   - оценка ёмкости предложения по компаниям и категориям;
   - анализ уровня цен (min / mean / max);
   - графики по компаниям и категориям.

---

## 2. Стек

- **Python**: `requests`, `beautifulsoup4`, `pandas`
- **Excel**: сводные таблицы и диаграммы
